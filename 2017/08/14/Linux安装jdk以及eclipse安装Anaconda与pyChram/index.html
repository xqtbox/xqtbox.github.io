3 重启linux

配置好PATH后，可以通过which conda或conda –version命令检查是否正确。

输入conda list 就可以查询，你现在安装了哪些库，常用的numpy, scipy名列其中。

如果你还有什么包没有安装上，可以运行conda install * 来进行安装，

如果某个包版本不是最新的，运行 conda update * 就可以了。

4 验证
